## Description
This note taker is an easy way to keep organized and on top of the day to day tasks that you are required to complete. It allows you to add, view save and delete notes as required.

## Usage
Copy the following URL and paste it into your brower address bar to go to the application.

https://stark-escarpment-55826.herokuapp.com/

Once you open the application you will be presented with an opening page as follows:

<img width="1274" alt="Index page" src="https://user-images.githubusercontent.com/120262482/229489694-c20f9a7d-6fa1-4d53-a5e0-feef19a25469.png">

Click the get started button and you will be brought to the Note Taking page,

<img width="1276" alt="Note Page" src="https://user-images.githubusercontent.com/120262482/229489906-e9a77c72-3c16-4415-9762-fe1ee89a7a34.png">

Any previously stored notes will be presented in the left column. When you are ready to start addin notes click on the "Note Title" and type in the title of your new note. Once the title has been entered either tab or click the note text area and type in the text you wish to add with the title. Once you click into the text area the "Save" icon will apprear at the top right of the page. Click the "Save" icon and the note will be stored on the left side of the page as shown.

<img width="1275" alt="First Note saved" src="https://user-images.githubusercontent.com/120262482/229490799-c3e2fbea-5bed-41db-804b-4f423afc7af7.png">

To delete the note click on the "Delete" icon beside the note title.

The following is a demonstration on how to use the Note Taker.

https://drive.google.com/file/d/1y5__EldA1cwaQspxlammGagOBZzHbj1Y/view
